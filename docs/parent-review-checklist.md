# Чек-лист для родителя

## Быстрая проверка тетради

1. В уравнениях есть последний шаг деления на коэффициент?
2. В неравенствах ответ записан как условие для `x`, а не как одно число?
3. Если делили на отрицательное число, знак неравенства перевёрнут?
4. В десятичных коэффициентах не потеряна целая часть числа?
5. В текстовой задаче первая строка объясняет, что такое `x`?
6. Ответ в текстовой задаче дан словами и проверен по смыслу?

## Как давать обратную связь

Формула: сначала отметить успех, потом один конкретный фокус.

Пример:

> Настюшик, у тебя хорошо получились скобки и таблицы в задачах. Сегодня главная точка — последний шаг в уравнении: если получилось `2x = 8`, обязательно делим на 2 и пишем `x = 4`.
