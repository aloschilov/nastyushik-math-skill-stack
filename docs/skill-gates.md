# Skill Gates

## Gate A: знаки

Навык считается закреплённым, если ребёнок решает 10 коротких примеров подряд без ошибки:

```text
7 - 12 + 5
-9 + 4 - 7
6 - 14 + 5
-10 + 7 - 6
```

## Gate B: десятичные коэффициенты

Закреплено, если 8 примеров подряд решаются без потери коэффициента и знака:

```text
12,5x - 1,2x - 0,8x
3,4x - 1,7x - 0,5x
-2,4b + 0,9b - 1,6b
```

## Gate C: уравнения

Закреплено, если в каждом решении явно есть последний шаг:

```text
2x = 8
x = 8 : 2
x = 4
```

## Gate D: особые уравнения

Нужно различать:

```text
0x = 0      -> любое число
0x = 5      -> нет решений
5x = 0      -> x = 0
```

## Gate E: неравенства

Закреплено, если ответ всегда записывается как условие:

```text
x > 7
x <= -5
```

При делении или умножении на отрицательное число знак меняется:

```text
-2x >= 10
x <= -5
```

## Gate F: текстовые задачи

Минимальный шаблон:

```text
Пусть x — это ...
Таблица: было / стало
Уравнение
Решение
Проверка по смыслу
Ответ словами
```
