# Повторяющиеся рабочие шаблоны prompts

Эти формулировки повторялись и могут использоваться дальше как шаблоны.

## Шаблон 1. Проверить день и сделать следующий комплект

```text
Проверь следующее задание. Сгенерируй pdf файл для ребёнка по результатам [номер] дня в смысле именно оценки навыков и обратной связи. Сгенерируй ещё следующий pdf файл с конкретными заданиями, чтобы можно было сразу выдать ребёнку. И отдельный файл с ответами и акцентами для проверки.
```

## Шаблон 2. Короткая версия после закрепления формата

```text
Аналогично для следующего дня.
```

## Шаблон 3. После экзамена, когда нужно менять акценты

```text
Аналогично для следующего дня. Учитывая что экзамен уже прошёл с неизвестным результатом, то выстраиваем акценты по основным аспектам непонимания.
```

## Шаблон 4. Только обратная связь, без новых заданий

```text
Уточнение, что фактически было решено. Времени было [время], постановки задачи нет. Сгенерируй обратную связь для ребёнка и для родителя. Задания будут корректироваться уже после текущей итерации.
```

## Шаблон 5. PDF только для ребёнка по итогам дня


## Шаблон 6. PDF с заданиями и отдельными ответами

```text
Сгенерируй pdf файл с конкретными заданиями, чтобы можно было сразу выдать ребёнку. И отдельный файл с ответами и акцентами для проверки.
```
