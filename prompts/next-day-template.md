# Prompt template для следующего дня

```text
Аналогично для следующего дня.

Проверь приложенную работу Настюшика. Сформируй:
1. PDF с обратной связью для ребёнка по результатам дня.
2. PDF с обратной связью для родителя.
3. PDF с конкретными заданиями на следующий день, чтобы можно было сразу выдать ребёнку.
4. PDF с ответами и акцентами для проверки.

Учитывай текущую capability matrix:
- сильные стороны: скобки, уравнения, особые случаи, текстовые задачи через таблицу;
- текущие акценты: последний шаг в уравнении, десятичные коэффициенты со знаками, неравенства с отрицательным множителем, смешанные задания без подсказки типа;
- стиль: поддерживающий, обращаться к ребёнку как “Настюшик”.
```
